# How to Clean Data on the OptiSigns Android Stick

This article, it guides you on how to clear data on the Android Stick.

1. Go to the Setting

[Image: 37780903482131]

2. Click **Apps**
[Image: 37780946862355]

3. Click **OptiSigns**

[Image: 37780946867347]

4. Select Clear data

[Image: 37780946872723]

5. Select **OK**

[Image: 37780946875155]

6. Then you can click **Open** to relaunch the OptiSigns app

[Image: 37780903501843]
