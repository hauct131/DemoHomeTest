# Kiosk Setup Hub

### Welcome to the OptiKiosk Setup Help Center.

OptiKiosks make digital signage easy, reliable, and powerful. Each kiosk comes pre-configured for use with OptiSigns and can be managed from anywhere.

Below, you will find all of our documentation on setting up your OptiKiosk, as well as where to buy and how to troubleshoot issues.

| |
| --- |
| **IMPORTANT NOTE** |
| In order to use the OptiKiosk, you'll need an [**OptiSigns subscription**](https://www.optisigns.com/pricing). To use it as a display, you can use any plan. To use it in Kiosk mode, you’ll need our [**Engage plan**](engage-plan-faqs.md). |
